## Overview

AtlasLogs is an interactive world map application that allows users to track and visualize their travels across different cities. It provides an engaging way to log and display your travel history.

## Features

- Interactive world map using Leaflet
- City tracking and logging
- Travel history visualization
- User-friendly interface

## Tech Stack

- **Frontend**: React, React-DOM
- **Map Library**: Leaflet
- **Build Tool**: Vite
- **Styling**: CSS Modules

## Getting Started

```bash
git clone https://github.com/souravkhan12/AtlasLogs
cd AtlasLogs
npm install
npm run dev
```
