## Overview

The Web Oasis is a comprehensive hotel management application built with modern web technologies. It provides a complete solution for managing hotel operations including bookings, guest management, and room availability.

## Features

- User-friendly interface for hotel staff
- Real-time room availability tracking
- Guest management system
- Booking management
- Supabase-powered backend for data persistence

## Tech Stack

- **Frontend**: React, React-Query
- **Backend**: Supabase API
- **Styling**: Tailwind CSS
- **Framework**: Next.js

## Getting Started

```bash
git clone https://github.com/souravkhan12/HorizonStay
cd HorizonStay
npm install
npm run dev
```
